# Jingxi Guo | Studio

Personal portfolio website for Jingxi Guo — sound artist, sound engineer, and creative technologist.

**Live site:** [yedianyang.com](https://yedianyang.com)

## Tech Stack

- [Jekyll](https://jekyllrb.com/) static site generator
- Hosted on [GitHub Pages](https://pages.github.com/)
- SCSS for styling

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Site will be available at `http://localhost:4000`.

## Structure

```
├── _config.yml          # Site configuration
├── _data/settings.yml   # Menu, projects, social links, SEO
├── _layouts/            # Page templates
├── _includes/           # Reusable HTML components
├── _sass/               # Stylesheets
├── assets/              # Images and CSS
├── pages/               # About, Research, Contact pages
└── projects/            # Project pages (Art Work, Film)
```
